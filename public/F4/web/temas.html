多Que es React.JS

Es una libreria de JS la cual es Open Source (es decir su codigo esta disponible al publico y es modificable), React.JS es desarrollada por Facebook(de la misma manera como bootstrap es desarrollada por twitter), es altamente util para construir interfaces de usuario interactivas, actualmente es una libreria muy popular debido a que al implementarla en algun proyecto se pueden reducir los tiempos de desarrollo; La clave de react es que trabaja sobre el DOM para optimizar el rendimiento de las paginas(rendimiento en terminos de respuestas).

多Que es Node.JS

Es un entorno de desarrollo el cual se instala en un servidor y este mismo permite desarrollar programas en lenguaje JS los cuales posteriormente pueden ser ejecutados del lado del cliente/usuario que acceda a dicha pagina/aplicacion web; Es de codigo abierto y es mantenido por Google, se caracterisa por ser eficiente, estable y por reducir los tiempos de respuestas de las ejecuciones.

多Que es una API REST

Es una interfaz de programacion, la cual puede ser vista como un lenguaje intermedio capaz de comunicar un programa y otros, basicamente, sive de mediador entre las peticiones que pueda hacer un programa con otro, las API REST son altamente utiles pues manejan un lenguaje mas sencillo e intuitivo de usar que lo que puede resultar intentar ententer el codigo que un programa en uso; Una API REST puede hacer acciones como consultar, actualizar, eleminar o crear elementos (Todo esto dependiendo de las opciones previamente creadas/programadas)

.Grafica de ejemplo de como podria operar o funcionar una API REST - web dossetenta. de Fuente

多Que es TypeScrip

Es una lenguaje desarrollado por Microsoft con el fin de servir de conjunto de herraminetas para JS, dotandole de mas caracteristicas como nuevos tipos de datos y nuevas sintaxis para agregar variables, funciones, objetos, etc. Los programas de TypeScript luego de ser escritos se compilan y se optimizan para ser converito en lenguaje JS estandar (lo cual implica que su uso seria el mismo que el de cualquier otro codigo de JS.)
