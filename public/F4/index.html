¿Que es JavaScript

JavaScript es un lenguaje de programacion de alto nivel de tipo interpretado, esto quiere decir que depende de un programa intermedio que permita la su ejecucion, y para el caso puntual de JavaScript(en adelante JS) son los navegadores. Una caracteristica puntual de JS es que es el lenguaje de programacion de la web, es el lenguaje que dota las paginas web de operabilidad indiferentemente de la manera como se hayan creado, JS le da dinamismo e interactivilidad a las paginas; Su funcionamiento es principalmente del lado del usuario, aunque en la actualidad han surgido nuevas implementaciones que permiten ejecutar codigo de JS desde un servidor y que desde este se puedan correr los programas (Para mayor informacion sobre esto ultimo consulte la pestaña temas)
