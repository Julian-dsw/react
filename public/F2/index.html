Las Media Queries

Son una técnica que busca que al momento de aplicar las hojas de estilo estas lo hagan de acuerdo algunas condiciones (que generalmente suelen ser las dimensiones de pantalla del tipo de dispositivo usado para acceder). La sintaxis básica seria algo de la siguiente forma:

Las reglas se pueden anidar y existen múltiples tipos de media como los de screen, los de print y los de speech, también existen varios tipos de condicionales para usar como el tamaño de pantalla (ya sea ancho con width o alto con height, el ratio u la orientación). Este es un ejemplo en el cual se anidan varias condiciones y tipos de media
